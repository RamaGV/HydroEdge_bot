# HydroEdge_bot - Automatización Inteligente para Hidroponía

HydroEdge_bot es una solución automatizada diseñada para gestionar y monitorear sistemas de cultivo hidropónico. Este bot utiliza sensores para controlar diversos parámetros de crecimiento y puede integrarse con plataformas de mensajería para ofrecer actualizaciones en tiempo real.

## Características
- Monitoreo en tiempo real de parámetros ambientales (pH, temperatura, humedad, etc.)
- Automatización del riego y gestión de nutrientes
- Notificaciones a través de plataformas como WhatsApp
- Integración con MQTT para la comunicación con otros dispositivos IoT

## Tecnologías
- Python
- MQTT
- Selenium para automatización de WhatsApp
- Integración con sensores de pH y temperatura
